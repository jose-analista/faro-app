**# Faro Mobile**  
   
**Faro Mobile is an Android application built with Java for street performers, especially jugglers. The app helps performers discover safe locations, connect with the community, and organize their daily activities.**  
   
**## Features**  
   
**- 📍 Find performance spots**  
**- 🗺️ Interactive map**  
**- 👥 Community features**  
**- 💬 Real-time communication**  
**- 📅 Performance planning**  
**- 🔔 Push notifications**  
**- 👤 User authentication**  
**- 📊 Performance statistics**  
   
**## Tech Stack**  
   
**- Java**  
**- Android SDK**  
**- Android Studio**  
**- Firebase Authentication**  
**- Cloud Firestore**  
**- Firebase Cloud Messaging (FCM)**  
**- Google Maps SDK for Android**  
   
**## Project Status**

**## Installation**  
   
**```bash**  
**git clone https://github.com/TU_USUARIO/faro-mobile.git**  
**```**  
   
**1. Open the project in Android Studio.**  
**2. Sync Gradle dependencies.**  
**3. Connect the project to Firebase.**  
**4. Add your `google-services.json` file.**  
**5. Build and run the application.**  
   
**## Requirements**  
   
**- Android Studio**  
**- Android SDK**  
**- Firebase Project**  
**- Google Maps API Key**
